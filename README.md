## Description of the project
Hey everyone! 🎉 I wanted to share some exciting news with you all. This year, I've challenged myself to create 10 fun apps on top of the ones I develop for clients. 💻 I'm proud to announce that I've already created my first app using Flutter, Provider, SQLite, SharedReference, and Firebase. 🚀 The app helps people manage their finances by calculating expenses and income across different categories. 💰 It even provides statistics for your finances 📈 and a pin code feature for added security 🔒. The app is multilingual 🌎, adaptive 📱, responsive 🖥️, and has a super user-friendly interface 👍.
